# Quadratic Equation Solver

A simple Python program to solve quadratic equations of the form:
__`ax² + bx + c = 0`__

- Example : 
```bash
a: 1
b: -3
c: 2
Polynomial: x² - 3.00x + 2.00 = 0
Delta = 1.0000
Solutions: x₁ = 2.0000, x₂ = 1.0000
--------------------------------------------------
Exit (y/n): n
--------------------------------------------------
```
